{% trans "Instance" %}

• {% trans "How to launch an instance?" %}

{% blocktrans %}1. Go to 'Instances' panel{% endblocktrans %}

{% blocktrans %}2. Click 'Launch Instance' button{% endblocktrans %}

{% blocktrans %}3. Set the instance name(hostname), choose a flavor for the instance.{% endblocktrans %}

{% blocktrans %}4. Choose a Windows or Linux OS image for the instance to boot.{% endblocktrans %}

{% blocktrans %}5. Set the instance administrator(Linux sudoer user)'s username and password.{% endblocktrans %}

{% blocktrans %}6. Set the default endpoint for the instance, e.g. to enable Linux SSH endpoint will let you can access the instance via SSH tools.{% endblocktrans %}

{% blocktrans %}7. Create a cloud service or choose one for the instance to join in.{% endblocktrans %}
